## vue3.3新特性
### defineOptions
主要是用来定义Option选项，如  
```vue
<script setup>
defineOptions({
  name: 'app' // 名字之类的
})
</script>
```
### defineModel
1.首先需要引入  
```vue
<script>
import { defineOptions } from 'vue
</script>
```
2.vite.config.js进行配置
```javascript
export default defineConfig({
    plugins: [
        vue({
            script: {
                defineModel: true
            }
        })
    ]
})
```

### 新增泛型的支持
对defineprops的改进，新增泛型支持需要在script标签加上generic="T"
```ts
const defineProps = defineProps<{
name: T[]
}>
```

对于事件
```ts
const emit = defineEmits<{
'send': [name: string]
}>
```
