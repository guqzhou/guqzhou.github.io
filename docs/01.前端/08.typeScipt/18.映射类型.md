## 映射类型
基于旧类型创建新类型（对象类型），减少重复，提高开发效率

### 1.联合类型实例
1. 映射类型是基于索引签名类型的，也使用了[]
2. key in obj 表示key可以是obj的任意一个
3. 映射类型只能在类型别名中使用，不能在接口中使用

```ts
type Obj = 'x' | 'y'
type T = { X: number, y: number}

type Type = {
    [key in Obj]: number
}
```
