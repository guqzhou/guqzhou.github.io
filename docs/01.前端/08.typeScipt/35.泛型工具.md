## 泛型工具
### Partial
是一个泛型类型，用于将一个类型的所有属性变为可选。
```typescript
interface  User {
    name: String
    age: String
}

type PartialUsre  = Partial<User>

// 实现原理
type customPartial<T> = {
    [P in keyof T] : T[P]
}
```


### Required
是一个泛型类型，用于将一个类型的所有属性变为必选。
```typescript
interface User {
    name: String
    age: String
}

type RequiredUser = Required<User>

// 实现原理
type customRequired<T> = {
    [P in keyof T]-?: T[P]
}

type RequiredUser = customRequired<User>
```

### pick
用于从一个类型中选取指定的属性。  
```typescript
interface User {
    name: String
    age: String
}

type pickUser = Pick<User, 'name'>

// 实现原理
type customPick<T, K extends keyof T> = {
    [P in K]: T[P]
}


```


### Exclude
用于从一个类型的属性集合中排除指定的属性。
原理：为什么要搞never？ 因为never在联合类型中会被忽略

```typescript
interface User {
    name: String
    age: String
}
type ExcludeUsre = Exclude<'A' | 'B' | 'C', 'A' | 'B'>

// 实现原理
type CustomUsre<T,K> = T extends K ? never : T
```


### Omit
用于创建一个新类型，该新类型从原始类型中排除指定的属性。
```typescript
interface User {
    name: String
    age: String
}
type  OmitUsre = Omit<User, 'age'>

type customOmit<T, K> = Pick<T, Exclude<keyof T, K>>

```
