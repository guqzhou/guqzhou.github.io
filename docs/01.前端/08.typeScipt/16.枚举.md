## 枚举
### 数字枚举
```ts
enum  Color  {
    red,
    blue,
    green
}
console.log(Color.red) // 0
```

### 增长枚举
```ts
enum Number {
    red = 1,
    blue,
    green
}
```

### const
const 声明的枚举会被编译成常量  
普通声明的枚举编译完后是个对象

### 反向枚举
```ts
enum Enum {
   fall
}
let a = Enum.fall;
console.log(a); //0
let nameOfA = Enum[a]; 
console.log(nameOfA); //fall
```
