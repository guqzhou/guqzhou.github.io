## 
### 联合类型
```ts
let ophone: number | string = 13989898
//  强转
!0  // true
!!0 // false
!1  // false
!!1 // true
```

### 交叉类型
```ts
interface man {
    name: string
}
interface Person {
    sex: number
} 
const  fun = (type: man & person) => {
    console.log(666)
}   12
```


### 类型断言
```ts
type as number
```
