# 常用 IP 查询与分析工具

整理了一些常用的 IP 查询、地理位置定位、隐私检测及网络分析工具，方便快速使用。

## 📍 基础 IP 查询 (常用)

适合快速查看本机外网 IP 和基本地理位置。

- **[Ping0.cc](http://ping0.cc/)**
  - IP 信息及欺诈风险评估工具。
- **[WhatIsMyIP.com](https://www.whatismyip.com/)**
  - 老牌 IP 查询网站，显示 IP、ISP 和城市信息。
- **[WhatIsMyIPAddress.com](https://whatismyipaddress.com/)**
  - 提供详细的 IP 地理位置映射和 ISP 信息。
- **[IP138](https://www.ip138.com/)**
  - 国内老牌查询网站，查询 IP、手机号归属地等。
- **[IPIP.net](https://www.ipip.net/)**
  - 国内专业的 IP 地理位置数据库，精度较高，提供多种工具。

## 🛡️ 隐私与安全检测

查看 IP 是否泄漏隐私，检测 VPN/代理状态及浏览器指纹。

- **[Whoer.net](https://whoer.net/)**
  - 检测伪装程度（百分比显示），检查 DNS 泄漏、WebRTC 泄漏、系统时间等指纹信息。
- **[BrowserLeaks (IP)](https://browserleaks.com/ip)**
  - 详细的浏览器泄漏检测，包括 WebRTC、Canvas 指纹等。
- **[IPhub](https://iphub.info/)**
  - 判断 IP 是否为代理（Proxy）、VPN 或数据中心 IP。

## 👨‍💻 开发者与高级分析

适合开发人员、网络工程师进行详细分析。

- **[ipinfo.io](https://ipinfo.io/)**
  - 提供 API 服务，显示 ASN、公司信息、托管商详情。
- **[IP2Location](https://www.ip2location.com/)**
  - 提供详细的全球 IP 数据库，支持经纬度、运营商等查询。
- **[BGPView](https://bgpview.io/)**
  - 查看 ASN（自治系统号）、BGP 前缀和网络路由图，适合网络调试。
- **[Ping.pe](http://ping.pe/)**
  - 全球多节点 Ping 测试，检查 IP 在全球的连通性（适合检测服务器是否被墙）。
- **[ITDOG (IT 狗)](https://www.itdog.cn/ping)**
  - 在线 Ping、TCPing 路由追踪工具，节点丰富，适合国内网络调试。

## 🔧 其他实用工具

- **[Chinaz 站长工具 - IP 查询](https://ip.tool.chinaz.com/)**
  - 综合性的站长工具，包含 Ping、Whois 等多种查询。
- **[WhoisCX](https://whoiscx.com/)**
  - 简洁高效的 Whois 查询与 IP 信息查询工具。
