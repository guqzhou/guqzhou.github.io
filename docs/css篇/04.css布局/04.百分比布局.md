## 百分比布局（流式布局）
### 概述
高度固定，宽度自适应

#### Demo演示
> 可用<kbd>F12</kbd>开发者工具查看元素及样式，可打开codepen在线编辑代码。
::: demo [vanilla]
```html
<html>
<ul class="demo-father">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
</ul>
</html>
<style>
    .demo-father {
        height: 200px;
    }
    .demo-father li{
        float: left;
        list-style: none;
        width: 25%;
        background-color: greenyellow;
    }
</style>
```
:::
